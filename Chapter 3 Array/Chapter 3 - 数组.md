## Chapter 3 数组

### 数组

- 将一个或者多个相同类型的对象连在一起所形成的类型

- ```C++
  int a; // int 类型
  int b[10] //int[10] 类型
  ```

- 数组初始化方式：

  - 缺省初始化

    ```c++
    int b[3];
    ```

  - 聚合初始化

    ```C++
    int b[3] = {1,2,3};
    int b[3] = {1,2}; // 第三个元素默认为0
    int b[] = {1,2,3}; // int[3]
    ```

- 不能用auto声明数组类型

- 数组不能复制

- 元素个数必须是一个常量表达式（编译期可计算的值）

- 字符串数组的特殊性

  - ```c++
    char str[] = "Hello"; //简化初始化，长度为n+1，结尾是'\0'表示结束
    char str[] = {'H','e','l','l','o'};
    
    ```
