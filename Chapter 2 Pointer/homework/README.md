## 第三章作业-基本运算题目生成器

### 文件结构

工程文件在VSCode的C++环境下开发完成，包括以下文件：

- `proj1.cpp`：主程序文件；
- `launch.json`：VSCode的C++环境配置文件；
- `tasks.json`：VSCode的C++环境配置文件；
- `Result.jpg`：输出结果截图；
- `README.pdf`：作业说明

运行`proj1.cpp`即可执行。

### 代码功能

实现以下功能：

- 随机数生成范围限制
- 加减乘除生成功能
- 随机数种子随机
- 除法结果小数限制2位
- 扩展1：
  - 一般模式与挑战模式切换交互（是否包含乘除法）
  - 题目数量交互（题目数量最大值模拟200，可修改）
  - 运算数据范围可交互（增加取值区间正确性判断）
- 扩展2：
  - 实现平均用时统计（使用ctime，但chrono更好）
  - 实现最短用时统计（使用ctime）
